1, 3 paramétert szed be:
    size - méret, default 50, minimum 50
    players - tribe-ok száma (1-4 között)
    init_cells - kezdő cellák tribe-onként, default 10, max 20

2, elhelyezés a következők szerint
    1. játékos -> (size/4, size/4)
    2. játékos -> (size/4 * 3, size/4)
    3. játékos -> (size/4, size/4 * 3)
    4. játékos -> (size/4 * 3, size/4 * 3)

    Ezek körüli 10-es sávban random init_cells-ek elhelyezése

    Cellák állapota: 0 -> üres, 1-2-3-4 -> élő, tribe-hoz tartozik

3, next step számítás:
    - összes élő cellán végig menni és összegyűjteni a vizsgálathoz:
        - élő céllákat
        - élő cellák szomszédait
        duplikációkat kiszűrni!
    - ezeken belül végimenni:
        - élőkön
            -> ha 0 vagy 1 élő szomszédja van -> meghal
            -> ha 2 vagy 3 élő szomszédja van -> él
            -> ha 3-nál több szomszédja van -> meghal
        - nem élőkön:
            -> ha 3 élő szomszédja van úgy, hogy csak 1 tribe-ból -> él
     - csak az élőket (életben maradtakat és feltámadókat) tovább vinni és válaszként átadni
